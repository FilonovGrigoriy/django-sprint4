# Blogicum

Учебный проект на Django: небольшой блог с публикациями, категориями, профилями пользователей и комментариями.
Реализованы: пагинация, отложенные публикации, загрузка изображений к постам, CRUD для постов/комментариев,
кастомные страницы ошибок и отправка писем через файловый backend.
